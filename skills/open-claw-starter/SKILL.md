# OpenClaw Starter Skill

Use this folder as a template for new skills.

## Customize

- Rename the skill in the frontmatter.
- Replace this description with the behavior you want the skill to provide.
- Point `config` at a skill-specific configuration file when needed.

## Register

Add the skill folder to `skills/MANIFEST.json` so it can be enabled or disabled centrally.
